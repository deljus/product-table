
# Поиск товаров + авторизация

Проект выполнен на стеке react, typescript, tailwindcss, vite


## Установка

Скачайте репозиторий и установите зависимости

```bash
  git clone https://github.com/deljus/product-table.git
  npm install
  npm run dev
```
